Elemento Explicación Nomenclatura

CSS Elements

Elemento Explicación Nomenclatura
Element Selector que refiere a un tipo de elemento HTML. p [color: green;]
Class Un selector para aplicar estilos a múltiples elementos .class [color: red;]
ID Un selector para aplicar estilos a un solo elemento con identificador único #id [color: blue;]
